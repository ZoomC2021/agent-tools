# Roo Code

Active contributors: zmang

## Purpose

Roo Code editor support provides slash commands for the Roo Code VS Code extension (formerly Cline).

## Directory layout

```
prompts/roocode/
├── create-pr.md
├── handoff.md
├── pr-reviewer-only.md
├── pr-reviewer.md
├── refactor.md
├── review.md
└── ultrareview.md
```

## Workflows supported

| Workflow | File |
|----------|------|
| create-pr | `create-pr.md` |
| handoff | `handoff.md` |
| pr-reviewer-only | `pr-reviewer-only.md` |
| pr-reviewer | `pr-reviewer.md` |
| refactor | `refactor.md` |
| review | `review.md` |
| ultrareview | `ultrareview.md` |

## Installation path

```
~/.roocode/
```

## Usage

In Roo Code, use the slash commands available in the chat interface.
