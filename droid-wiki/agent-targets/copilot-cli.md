# Copilot CLI

Active contributors: zmang

## Purpose

Copilot CLI support provides slash commands for the GitHub Copilot command-line interface.

## Directory layout

```
prompts/copilot-cli/
├── create-pr.md
├── deslop.md
├── handoff.md
├── pr-reviewer-only.md
├── pr-reviewer.md
├── refactor.md
├── review.md
└── ultrareview.md
```

## Workflows supported

| Workflow | File |
|----------|------|
| create-pr | `create-pr.md` |
| deslop | `deslop.md` |
| handoff | `handoff.md` |
| pr-reviewer-only | `pr-reviewer-only.md` |
| pr-reviewer | `pr-reviewer.md` |
| refactor | `refactor.md` |

| ultrareview | `ultrareview.md` |

## Installation path

```
~/.github/copilot/
```

## Usage

In Copilot CLI, use the commands with the `gh copilot` interface.
