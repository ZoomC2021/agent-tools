# Claude Code

Active contributors: zmang

## Purpose

Claude Code support provides slash commands for the Claude CLI tool.

## Directory layout

```
prompts/claude/
├── create-pr.md
├── deslop.md
├── handoff.md
├── predict-issues.md
├── pr-reviewer-only.md
├── pr-reviewer.md
├── refactor.md
├── review.md
└── ultrareview.md
```

## Workflows supported

| Workflow | File |
|----------|------|
| create-pr | `create-pr.md` |
| deslop | `deslop.md` |
| handoff | `handoff.md` |
| predict-issues | `predict-issues.md` |
| pr-reviewer-only | `pr-reviewer-only.md` |
| pr-reviewer | `pr-reviewer.md` |
| refactor | `refactor.md` |
| review | `review.md` |
| ultrareview | `ultrareview.md` |

## Installation path

```
~/.claude/commands/
```

## Usage

In Claude Code, type `/` to see available slash commands, or type the command name directly.
