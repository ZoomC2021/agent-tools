# Windsurf

Active contributors: zmang

## Purpose

Windsurf editor support provides AI workflows for the Windsurf IDE, using SKILL.md format organized in subdirectories.

## Directory layout

```
prompts/windsurf/
├── create-pr/
│   └── SKILL.md
├── handoff/
│   └── SKILL.md
├── pr-reviewer/
│   └── SKILL.md
├── refactor/
│   └── SKILL.md
├── review/
│   └── SKILL.md
└── ultrareview/
    └── SKILL.md
```

## Workflows supported

| Workflow | Path |
|----------|------|
| create-pr | `create-pr/SKILL.md` |
| handoff | `handoff/SKILL.md` |
| pr-reviewer | `pr-reviewer/SKILL.md` |
| refactor | `refactor/SKILL.md` |
| review | `review/SKILL.md` |
| ultrareview | `ultrareview/SKILL.md` |

## Installation path

```
~/.windsurf/
```

## Usage

In Windsurf, skills are available through the Cascade AI interface.
